# Task Manager Web App

A simple full-stack Task Manager web application built using **Flask**, **SQLite**, **HTML**, **CSS**, and a bit of **JavaScript**.

This project is being developed step by step to learn how frontend and backend work together in a real-world application.

---

## Features

- Add new tasks
- Delete tasks
- Edit existing tasks
- Mark tasks as completed
- Persistent storage using SQLite
- Dark mode with saved user preference
- Clean and simple UI

---

## Tech Stack

**Frontend**
- HTML
- CSS
- JavaScript

**Backend**
- Python
- Flask

**Database**
- SQLite

---

## Install depedencies
pip install flask

## Run application using
python app.py

## Open in browser using
http://127.0.0.1:5000

## Dark Mode
The app includes a dark mode toggle.
The selected theme is saved using localStorage and persists across sessions.
